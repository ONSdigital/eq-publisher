# eq-publisher
An API for publishing [eq-author](http://github.com/ONSDigital/eq-author) questionnaires.

## Installation

To install dependencies, simply run:
```
yarn install
```

To run the application:
```
yarn start
```

## Testing

To run all tests:
```
yarn test
```
